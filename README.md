# Human Face AI

Vite + React + TypeScript 製の GitHub Pages です。

- [GitHub Pages - Human Face AI](https://mkrydik.github.io/human-face-ai/)

人面瘡に話しかけてみてください。


## インストール

```bash
$ npm install
```


## 開発

```bash
$ npm run dev
```

- `http://localhost:5173/human-face-ai/`


## ビルド

```bash
$ npm run build
$ npm run serve
```

- `http://localhost:4173/human-face-ai/`
